Тема проєкту: Платформа AirBnB
Ця робота представляє візуальну модель системи оренди нерухомості за допомогою чотирьох типів UML-діаграм.

### Короткий опис діаграм:
1. **Use Case Diagram**: Відображає реєстрацію нового учасника та пошук житла за параметрами.
2. **Class Diagram**: Визначає структуру об'єктів: власники, квартири, фотографії, відгуки з оцінками та бронювання.
3. **Activity Diagram**: Моделює процес подачі заявки на резервування та прийняття рішення власником (підтвердження або відмова).
4. **Sequence Diagram**: Візуалізує послідовність дій у часі, включаючи списання коштів сервісом та виплату власнику після заїзду.
